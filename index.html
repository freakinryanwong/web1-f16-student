WHY WEB DESIGN?

for the longest time, web design would be my least favorite thing to do—only because i didn't know how to do it. at my internship, it actually became a joke because of how vocal i am about my distaste when we were creating splash pages for small events. and the audience would always react hella well to the new websites. so yeah, web design is definitely a very demanding and important skill to have in this day and age. so just to make sure i'm ready for the outside world after graduation, i expect to be able to create the slickest most beautiful website that will make andrew ringler cry.
